# LawMind Agent 转型 — 复杂法律问题测试集

> 用途：对比评估传统模式与 Agent 模式的回答质量。
> 每个问题需要至少 2 个 Tool 调用才能完整回答。
> 评估维度：完整性(1-5)、准确性(1-5)、幻觉检查(PASS/FAIL)、耗时(秒)。

## 一、劳动纠纷场景（4 题）

### Q1：违法辞退赔偿计算

**问题**：我在公司工作了3年半，月薪9000元，上周公司突然通知我被辞退，没有给任何书面理由。我可以要求多少赔偿？应该怎么维权？

**预期 Tool 调用**：classifyLegalIntent → expandLegalQuery → searchLawKnowledge（劳动合同法第47/48/87条）→ 金额计算

**标准答案要点**：
- [ ] 判断为违法解除劳动合同（未说明理由、无书面通知）
- [ ] 引用《劳动合同法》第47条：经济补偿按工作年限，每满一年支付一个月工资
- [ ] 引用《劳动合同法》第48条：违法解除需支付双倍赔偿
- [ ] 引用《劳动合同法》第87条：赔偿金=经济补偿×2
- [ ] 正确计算：9000 × 4 × 2 = 72000元（工作3年半按4年算）
- [ ] 建议维权路径：协商 → 劳动仲裁（1年内）→ 诉讼

### Q2：工伤认定与赔偿

**问题**：我在建筑工地干活时从脚手架摔下来，导致左腿骨折，住院花了3万多。老板没有给我交社保，也没有签劳动合同。我能认定为工伤吗？能获得哪些赔偿？

**预期 Tool 调用**：classifyLegalIntent → expandLegalQuery → searchLawKnowledge（工伤保险条例/劳动合同法）→ searchSimilarQuestions

**标准答案要点**：
- [ ] 指出事实劳动关系同样受工伤保护（《劳动合同法》第7条）
- [ ] 引用《工伤保险条例》第14条：工作时间工作场所因工作原因受伤属于工伤
- [ ] 说明未缴社保时由用人单位承担全部工伤待遇
- [ ] 列出赔偿项目：医疗费、停工留薪期工资、护理费、一次性伤残补助金等
- [ ] 提醒：需在受伤之日起1年内申请工伤认定
- [ ] 建议收集证据：工资记录、工友证言、微信聊天记录、医院病历

### Q3：加班费争议

**问题**：我在一家互联网公司做程序员，合同签的是标准工时制，月薪15000元。但过去一年几乎每天加班到晚上9点，周末也经常加班，公司从不给加班费，说"我们实行的是弹性工作制"。我可以要求补发加班费吗？大约能要回多少？

**预期 Tool 调用**：classifyLegalIntent → expandLegalQuery → searchLawKnowledge（劳动法关于工作时间的条款）→ 金额计算

**标准答案要点**：
- [ ] 引用《劳动法》第36条：每日工作不超过8小时，每周不超过44小时
- [ ] 引用《劳动法》第44条：加班费标准（平日1.5倍、休息日2倍、法定假日3倍）
- [ ] 指出"弹性工作制"不等于免除加班费义务
- [ ] 提醒举证责任：劳动者需证明加班事实（考勤记录、工作邮件、钉钉打卡等）
- [ ] 金额估算示范：按工作日每天加班3小时、周末每月2天估算年度加班费
- [ ] 建议：先协商 → 劳动监察投诉 → 劳动仲裁

### Q4：竞业限制纠纷

**问题**：我从原公司离职时签了竞业限制协议，约定2年内不得从事同行业工作，每月补偿金2000元。但离职后公司只给了3个月补偿金就停了。我找到了一家同行业公司，月薪比原来高50%，但担心被起诉。现在竞业限制协议还有效吗？我能不能去新公司？

**预期 Tool 调用**：classifyLegalIntent → searchLawKnowledge（劳动合同法第23/24条 + 竞业限制司法解释）→ verifyCitation

**标准答案要点**：
- [ ] 引用《劳动合同法》第23条：竞业限制需支付经济补偿
- [ ] 引用司法解释：用人单位3个月未支付补偿金，劳动者可请求解除竞业限制
- [ ] 分析：原公司已停付超过3个月，你有权解除竞业限制
- [ ] 建议：书面通知原公司解除竞业限制协议，保留证据
- [ ] 提醒：可要求原公司补发拖欠的补偿金
- [ ] 注意：竞业限制范围不得扩大至无关行业

## 二、婚姻家庭场景（3 题）

### Q5：离婚财产分割

**问题**：我和妻子结婚8年，婚后买了一套房（首付我家出了60万，她家出了20万，贷款共同还），现在房产价值300万，还有50万贷款没还。另外我婚前有一笔50万的股票投资，婚后增值到了80万。如果离婚，房产和股票怎么分？

**预期 Tool 调用**：classifyLegalIntent → expandLegalQuery → searchLawKnowledge（民法典婚姻家庭编）→ 金额计算

**标准答案要点**：
- [ ] 引用《民法典》第1062条：婚后购买的房产属于夫妻共同财产
- [ ] 引用《民法典》第1087条：共同财产分割原则（协议优先，照顾子女和女方）
- [ ] 房产计算：净值=300万-50万=250万，原则上各得125万（首付出资比例可协商调整）
- [ ] 引用《民法典》第1063条：婚前财产属于个人
- [ ] 股票分析：50万本金为婚前个人财产，30万增值部分若为自然增值则仍属个人
- [ ] 注意区分：婚后投入了劳动管理的增值可能被认定为共同财产

### Q6：子女抚养权与抚养费

**问题**：我和前夫离婚2年了，孩子今年8岁，当时协议孩子跟他，我每月付1500元抚养费。但现在我发现他长期在外地工作，孩子实际由爷爷奶奶照顾，学习成绩也下降了。我想变更抚养权，并且让他承担抚养费。可行吗？流程是怎样的？

**预期 Tool 调用**：classifyLegalIntent → searchLawKnowledge（民法典婚姻家庭编关于抚养权的条款）→ searchSimilarQuestions

**标准答案要点**：
- [ ] 引用《民法典》第1084条：8岁以上子女应尊重其真实意愿
- [ ] 变更抚养权的法定情形：抚养方不尽义务、对子女身心健康不利等
- [ ] 指出对方长期不在孩子身边可能构成不尽抚养义务
- [ ] 抚养费标准：月总收入的20%-30%，参考当地生活水平
- [ ] 建议流程：协商 → 调解 → 向法院起诉变更抚养权
- [ ] 提醒：可收集孩子成绩变化、对方出差记录等证据

### Q7：婚前债务与婚后财产

**问题**：我丈夫婚前为了买房借了亲戚30万，婚后我们俩一起还了15万。现在我们要离婚，他说这30万是夫妻共同债务，要求我承担一半。另外，他婚前买的房婚后用我们共同收入还贷，房子的增值部分我能分吗？

**预期 Tool 调用**：classifyLegalIntent → searchLawKnowledge（民法典关于夫妻债务的条款 + 婚姻家庭编司法解释）→ verifyCitation

**标准答案要点**：
- [ ] 引用《民法典》第1064条：婚前一方所负债务原则上为个人债务
- [ ] 婚前借款用于购买婚前个人房产，属于个人债务
- [ ] 婚后用共同收入偿还的15万，你可主张一半（7.5万）的补偿
- [ ] 引用婚姻家庭编司法解释：婚后共同还贷部分及对应增值属于共同财产
- [ ] 房产增值计算公式：增值部分 × (共同还贷额÷总购房成本) ÷ 2
- [ ] 注意：债权人（亲戚）可能将两人共同作为被告

## 三、合同纠纷场景（3 题）

### Q8：消费欺诈与惩罚性赔偿

**问题**：我在某电商平台买了一部iPhone，页面明确标注"全新正品、官方质保"，但收到后发现是翻新机，已经激活超过半年了。平台客服说"可以退货退款，承担运费"。我觉得这是欺诈，能不能要求更高赔偿？应该怎么操作？

**预期 Tool 调用**：classifyLegalIntent → searchLawKnowledge（消费者权益保护法 + 电子商务法）→ verifyCitation

**标准答案要点**：
- [ ] 引用《消费者权益保护法》第55条：欺诈行为退一赔三，不足500元按500元
- [ ] 分析"全新正品"标注与实际"翻新机"构成欺诈
- [ ] "退一赔三"计算：退还手机款 + 赔偿手机款×3（即共获得4倍手机款）
- [ ] 引用《电子商务法》第38条：平台知道商家侵权未采取措施的承担连带责任
- [ ] 建议操作：固定证据（截图、聊天记录、鉴定报告）→ 12315投诉 → 向平台索赔 → 诉讼
- [ ] 提醒：不要只接受退款退货，那等于放弃了三倍赔偿权利

### Q9：合同违约与定金罚则

**问题**：我定了一套婚房，签了购房合同交了20万定金，约定3个月后交房。但现在开发商说因为"政策原因"要延期至少半年，还不给具体日期。我觉得他们可能资金链出问题了，想退房但开发商说退房算我违约，定金不退。定金能要回来吗？能要求双倍返还吗？

**预期 Tool 调用**：classifyLegalIntent → expandLegalQuery → searchLawKnowledge（民法典合同编）→ searchSimilarQuestions

**标准答案要点**：
- [ ] 引用《民法典》第587条：收受定金方违约应双倍返还
- [ ] 分析：开发商延期交房构成违约，不是买方违约
- [ ] 引用《民法典》第586条：定金不得超过合同标的额20%，超出部分无效
- [ ] 定金罚则：开发商违约应双倍返还→应返还40万
- [ ] 如定金超过20%标准，超出部分按预付款处理（原额返还）
- [ ] 建议：书面催告开发商 → 发解除合同通知 → 协商不成则起诉

### Q10：民间借贷利息与担保

**问题**：朋友向我借了30万做生意，写了借条，约定年利率24%，他表弟在借条上签了"担保人"三个字。现在到期一年了，朋友说生意亏了没钱还，他表弟说"我只是见证人不是担保人"。我想起诉他们俩，本金和利息能全要回来吗？法律支持的最有利率是多少？

**预期 Tool 调用**：classifyLegalIntent → searchLawKnowledge（民法典合同编 + 民间借贷司法解释）→ verifyCitation → 金额计算

**标准答案要点**：
- [ ] 引用民间借贷司法解释：利率上限为合同成立时一年期LPR的4倍（当前约13.8%-14.8%）
- [ ] 24%年利率超过法定上限，超出部分法院不支持
- [ ] 合法利息计算：30万 × LPR×4 × 1年（按实际LPR计算）
- [ ] 引用《民法典》第686条：担保方式不明确时按一般保证处理
- [ ] 一般保证 vs 连带保证的区别（一般保证有先诉抗辩权）
- [ ] 提醒诉讼时效：还款到期日起3年内
- [ ] 注意：民间借贷需有转账记录等实际交付证明
